# TradingView Demos

Галерея демонстраций всех вариантов создания графиков с TradingView Charting Library и Lightweight Charts.

## Структура

```
tradingview-demos/
├── chart-types/        # 5 типов создания графиков
├── series-types/       # 7 типов серий данных
├── conditions/        # 8 демо по условиям/темам
└── index.html         # Галерея всех вариантов
```

## Запуск

```bash
cd tradingview-demos
python -m http.server 8080
```

Открыть: http://localhost:8080
