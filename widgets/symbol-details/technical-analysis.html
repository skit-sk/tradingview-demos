Technical Analysis Widget

Технический анализ символа: осцилляторы и Moving Averages.

Код для встраивания:

<div id="widget-container"></div>
<script type="text/javascript" src="https://s3.tradingview.com/tv.js"></script>
<script type="text/javascript">
new TradingView.widget({
    "width": "100%",
    "height": 450,
    "locale": "en",
    "darkMode": true,
    "symbol": "NASDAQ:AAPL",
    "isTransparent": false,
    "showIntervalUI": true,
    "showSymbolLogo": true,
    "showToolbar": true
});
</script>
