← Gallery 2. Yield Curve Chart
Synthetic Data

Method

createYieldCurveChart()

Description

График кривой доходности для облигаций и процентных ставок. Горизонтальная шкала — линейная (месяцы/duration units). X-ось = срок погашения в месяцах.

Parameters

ParameterValueDescription
layout.background'#131722'Цвет фона графика
layout.textColor'#d1d4dc'Цвет текста на графике
series.color'#2962FF'Цвет линии кривой доходности
series.lineWidth2Толщина линии кривой
yieldCurve.baseResolution1Базовое разрешение временной шкалы (1 месяц)
yieldCurve.minimumTimeRange10Минимальный диапазон шкалы времени (месяцев)
yieldCurve.startTimeRange3Начальный диапазон времени (стартовая точка)
handleScrollfalseОтключить прокрутку графика (фиксированный вид)
handleScalefalseОтключить масштабирование графика (фиксированный вид)

Data Source

Synthetic yield curve data: US Treasury yields by maturity (months)

Input Format

LineData: { time: months, value: interest_rate_% }
