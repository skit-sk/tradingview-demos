# Fixed Size

## Когда использовать

Когда нужен график с **точно заданными размерами** — не адаптируется под контейнер.

## Ключевые параметры

```javascript
const chart = createChart(container, {
  width: 800,      // фиксированная ширина в px
  height: 400,     // фиксированная высота в px
  autoSize: false, // по умолчанию
});
```

## Особенности

- Размер не меняется при изменении окна браузера
- Контейнер должен иметь достаточный размер для отображения
- Можно создать несколько графиков разного размера на одной странице
