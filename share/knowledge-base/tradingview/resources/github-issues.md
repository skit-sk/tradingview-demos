## Как создать issue

### Перед созданием

1. Проверьте existing issues на дубликаты
2. Обновите до последней версии библиотеки
3. Подготовьте минимальный reproducible пример

### Шаблон issue

```markdown
## Bug Description
[Описание проблемы]

## Steps to Reproduce
1. [Шаг 1]
2. [Шаг 2]
3. [Шаг 3]

## Expected Behavior
[Что ожидалось]

## Actual Behavior
[Что произошло]

## Environment
- Library version: [версия]
- Browser: [браузер]
- OS: [OS]

## Code
```javascript
// Минимальный код для воспроизведения
```
```

## Известные issues

Проверьте pinned issues и labels:
- `bug` — подтверждённые баги
- `enhancement` — запросы на фичи
- `help wanted` — нужна помощь
- `good first issue` — для новичков
